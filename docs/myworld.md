## Quick Start

### 长久以来，在我脑子里思索的很多东西，一直由于种种原因无法付诸行动。即使有所行动的事，也没有见到很好的结果。随着时间推移，一些想法逐渐过时而淘汰，而另一些想法则被他人不谋而合地逐渐实现。随着年纪的增长，觉得有必要将所思所想写下来，和别人探讨，寻求志同道合者，一起将理想变为现实；如果自己无法亲身实践，能让别人所了解，那怕是能给别人以一些启发，也足以欣慰！
